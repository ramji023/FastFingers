## ğŸ® How It Works

1. **Start Test**: Click start or begin typing to initiate the test
2. **Type Away**: Type the displayed text as quickly and accurately as possible
3. **View Results**: After completion, see detailed statistics and performance graphs

### Metrics Calculated:

- **WPM (Words Per Minute)**: Standard measure of typing speed
- **Accuracy**: Percentage of correctly typed characters
- **Performance Graph**: Visual representation of speed and accuracy over time
